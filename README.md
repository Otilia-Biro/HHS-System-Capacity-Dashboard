# System Capacity & Care Load Analytics Dashboard

## Project Overview
This project analyzes operational demand and system capacity trends in a care system dataset (mid-2023 to mid-2025) using Python and Streamlit.

The dashboard provides:
- KPI metrics (Mean, Min, Max, Sum)
- Interactive visualizations
- Distribution analysis (histograms)
- Outlier detection (IQR method)
- Summary statistics for operational insights

## Tools & Technologies
- Python (Pandas, Matplotlib)
- Streamlit
- Data Cleaning & KPI Engineering

## Dataset
Cleaned dataset used: cleaned_hhs_dataset.csv

## How to Run the Project Locally
1. Install requirements:
pip install -r requirements.txt

2. Run the dashboard:
streamlit run app.py
